# Sales Analysis using Python + SQLite

A simple project that creates a small SQLite database, runs basic SQL queries, and visualizes sales data using Python.

## Features
- Creates `sales_data.db` with a `sales` table
- Inserts sample sales records
- SQL queries for total quantity and revenue
- Displays results using Pandas
- Generates a bar chart (`sales_chart.png`)

## How to Run
```bash
pip install pandas matplotlib
python analysis.py
